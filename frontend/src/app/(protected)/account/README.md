# Account Feature

User profile and settings management.

## Route: `/dashboard/account`

User profile settings and billing information.
**Access**: Protected (requires authentication)

## Components To Build

- **ProfileForm** - Edit user profile (name, email, avatar)
- **ChangePasswordForm** - Update password
- **LinkedAccountsSection** - Manage OAuth linked accounts
- **DeleteAccountSection** - Account deletion with confirmation
- **EmailPreferences** - Notification settings (future)

## Features

- Update profile information
- Change password
- Link/unlink OAuth providers
- View account information
- Delete account with confirmation
