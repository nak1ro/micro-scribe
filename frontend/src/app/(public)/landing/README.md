# Landing Feature

Landing page sections and marketing content.

## Route: `/`

Large marketing site with sections accessible via fragment identifiers.
**Access**: Public (all users)

## Sections (Fragment Links)

- `#pricing` - Subscription tier comparison
- `#languages` - Supported languages showcase
- `#reviews` - User testimonials and social proof
- `#premium` - Pro plan features and benefits

## Components To Build

- **Hero** - Main hero section with CTA
- **PricingSection** - Free vs Pro plan comparison table
- **LanguagesSection** - Supported languages grid
- **ReviewsSection** - User testimonials carousel
- **PremiumSection** - Pro features showcase
- **FeaturesSection** - Core product features
- **CTASection** - Call-to-action with signup button
