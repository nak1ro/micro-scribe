# Common Components

Shared components used across multiple features.

## Contains

- **Logo** - Application logo in various sizes
- **ThemeToggle** - Dark/light mode switcher
- **ErrorBoundary** - React error boundary wrapper
- **LoadingScreen** - Full-page loading state
- **EmptyState** - Placeholder for empty lists/content
- **ConfirmDialog** - Reusable confirmation modal
- **Pagination** - List pagination controls

## Guidelines

- Feature-agnostic, used across the entire app
- Should not contain business logic
- May compose primitive UI components
