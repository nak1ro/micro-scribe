# Form Components

Reusable form elements and validation-aware inputs.

## Contains

- **FormField** - Wrapper with label, input, and error message
- **FormGroup** - Groups of related form fields
- **PasswordInput** - Password field with visibility toggle
- **FileUpload** - Drag-and-drop file upload component
- **CheckboxField** - Checkbox with label integration
- **RadioGroup** - Radio button groups
- **SelectField** - Styled select dropdowns

## Guidelines

- Integrate with form validation libraries (React Hook Form, Formik)
- Display validation errors consistently
- Support disabled and loading states
- Provide accessible labels and error announcements
