# Layout Components

Structural components that define page layouts and navigation.

## Contains

- **Header** - Top navigation bar with logo, nav links, user menu
- **Footer** - Site-wide footer with links and info
- **Sidebar** - Dashboard sidebar navigation
- **MainLayout** - Public pages layout wrapper
- **DashboardLayout** - Authenticated dashboard layout wrapper
- **PageContainer** - Consistent page padding and max-width

## Guidelines

- Handle responsive breakpoints
- Manage navigation state
- Provide consistent structure across pages
