# Library / Utilities

Shared utilities, helpers, and configuration.

## Structure

- `utils/` - Pure utility functions
- `constants/` - Application-wide constants
- `config/` - Environment and app configuration
- `validation/` - Form validation schemas (Zod, Yup)

## Guidelines

- Functions should be pure and testable
- No React dependencies in utility functions
- Use descriptive names for constants
- Type all configuration values
