# Constants

Application-wide constant values.

## Contains

- **routes.ts** - Route path constants
- **api-endpoints.ts** - API endpoint paths
- **plans.ts** - Subscription plan definitions
- **file-types.ts** - Supported file types and MIME types
- **limits.ts** - Upload size limits, duration limits

## Guidelines

- Use ALL_CAPS for primitive constants
- Use PascalCase for object/enum constants
- Group related constants in objects
